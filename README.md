# Portfolio Coming Soon Page

Portfolio "coming soon" landing page built with React.js and Tailwind CSS.

## Live Demo

[View Live Demo](https://portfolio-coming-soon.vercel.app) *(We'll update this after deployment)*

## Features

- Modern dark theme with yellow accents
- Live countdown timer
- Email notification form
- Fully responsive design
- Glass morphism effects
- Smooth animations

## Technologies Used

- React.js
- Tailwind CSS
- Vite
- Heroicons
